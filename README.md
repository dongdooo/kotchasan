# คชสาร (kotchasan) PHP Framework
ช้างนอกจากจะเป็นสัญลักษณ์ของ PHP แล้ว ยังเป็นสัญลักษณ์ประจำชาติของเราอีก ผมเลยเลือกที่จะใช้ชื่อนี้เป็นชื่อของ Framework ที่ออกแบบโดยคนไทย 100%

##คุณสมบัติ
* สถาปัตยกรรม HMVC ทำให้การเพิ่มหรือลดโมดูลเป็นไปโดยง่าย ไม่ขึ้นแก่กัน
* สนับสนุนการทำงานแบบหลายโปรเจ็ค
* ปฏิบัตตามมาตรฐาน PSR-1, PSR-2, PSR-4
* เป็น PHP Framework ที่ได้รับการ Optimize ทั้งทางด้านความเร็วและประสิทธิภาพในการทำงาน ตลอดจนการใช้หน่วยความจำ ให้มีประสิทธิภาพดีที่สุด ทำให้สามารถทำงานได้เร็วกว่า และรองรับผู้เยี่ยมชมพร้อมกันได้มากกว่า

## องค์ประกอบของ คชสาร
คชสารจะประกอบด้วยเฟรมเวอร์คหลัก 3 ตัว ที่ออกแบบเพื่อใช่ร่วมกัน ทั้งส่วนของ PHP, CSS และ Javascript
* Kotchasan PHP Framework
* GCSS CSS Framework
* GAjax Javascript Framework

## ความต้องการ
* PHP 5.3 ขึ้นไป

##เงื่อนไขการใช้งาน
* เป็น Open Source สามารถนำไปใช้งานได้ฟรี ไม่มีเงื่อนไข
* สามารถนำไปพัฒนาต่อยอดเป็นลิขสิทธิ์ของตัวเองได้ (เฉพาะส่วนที่พัฒนาเพิ่มเติมเอง)
